# Project Assets & Computer Vision Masks

This folder stores the visual data used for testing the image processing algorithms.

* **Original Scene:** `img_1.jpg` represents the raw top-down view of the arena.
* **Binary Masks:** Examples of the HSV segmentation results for different colors (robot identification and track limits).

> **Vision Strategy:** The system uses a global camera to eliminate cumulative errors typically found in pure wheel-encoder odometry.
