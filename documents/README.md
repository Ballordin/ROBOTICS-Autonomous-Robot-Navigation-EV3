# Documentation

This folder contains the complete technical report for the project.

## Highlights of the Report:
* **Kinematic Modeling:** Detailed mathematical derivation of the robot's movement.
* **Algorithm Flowcharts:** Step-by-step logic for the autonomous navigation and obstacle avoidance.
* **Vision Pipeline:** Explanation of the image segmentation process and coordinate transformation (pixels to millimeters).
* **Experimental Results:** Data on odometry error correction and sign recognition accuracy.
